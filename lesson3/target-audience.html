Happy Winds - Site Plan

Main Information

  • Site name: "Happy Winds"

  • The site's purpose is: To be a source of reliable information. On the site, the user can access accurate information about the local weather for free. The user can also access the weather in neighboring cities. It is relevant to the site's purpose that the user can access it wherever he may be. That implies the need for a responsive layout. It is also necessary the usage of easy-to-use user interfaces to allow users without much experience to be able to navigate the site fluently.
